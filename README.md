Project Description
MediLedger is a revolutionary blockchain-based medical records management system built on Ethereum. It provides a secure, transparent, and decentralized platform for managing patient medical records while ensuring privacy, data integrity, and controlled access between patients and healthcare providers.
The system leverages smart contract technology to create an immutable ledger of medical records, allowing patients to maintain complete control over their healthcare data while enabling seamless and secure sharing with authorized medical professionals.
Project Vision
Our vision is to transform healthcare data management by:

Empowering Patients: Giving patients complete ownership and control over their medical data
Enhancing Security: Utilizing blockchain technology to create tamper-proof medical records
Improving Accessibility: Enabling seamless and secure access to medical records across different healthcare providers
Reducing Costs: Eliminating intermediaries and reducing administrative overhead in healthcare data management
Fostering Innovation: Creating a foundation for advanced healthcare analytics and AI-driven medical insights

MediLedger aims to bridge the gap between traditional healthcare systems and modern blockchain technology, creating a future where medical data is secure, accessible, and truly owned by the patients themselves.

Getting Started
Prerequisites

Node.js v16 or higher
Hardhat or Truffle development environment
MetaMask or compatible Web3 wallet
Ethereum testnet access (Goerli, Sepolia)

Installation
bash# Clone the repository
git clone https://github.com/your-org/mediledger.git
cd mediledger

# Install dependencies
npm install

# Compile smart contracts
npx hardhat compile

# Run tests
npx hardhat test

# Deploy to testnet
npx hardhat run scripts/deploy.js --network goerli
Usage

Deploy the contract to your preferred Ethereum network
Set up the admin account for doctor verification
Register patients using the registerPatient function
Verify doctors through the admin account
Patients can grant access to doctors using grantAccess
Doctors can add medical records for authorized patients
